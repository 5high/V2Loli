# V2Loli

## Usage
* 1.Import the project to AndroidStudio
* 2.Change the Site Info in app/src/main/java/com/v2ray/loli/constant/HttpConfig.java
* 3.Change the TG Group in app/src/main/kotlin/com/v2ray/loli/AppConfig.kt 
* 4.Build and Run
* 5.Enjoy~
